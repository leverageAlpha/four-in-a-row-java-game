# Four in a Row Java Game

Java client-server implementation of a Four in a Row game with several custom game modes and a JavaFX interface.

## Overview

This collaborative academic project implements a playable board game inspired by Connect Four. The project extends the classic rules with additional modes and a client-server structure.

## Game Modes

- Classic mode: standard 7x6 four-in-a-row gameplay.
- Super pieces mode: special pieces introduce strategic effects.
- Pop Out mode: players can remove pieces from the bottom row.
- Five in a Row mode: expanded win condition and board setup.

## Tech Stack

- Java
- JavaFX
- Client-server architecture
- CSS for interface styling
- Eclipse project structure
